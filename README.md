# Interactive Flowchart Visualizer
A web application that automatically generates interactive flowchart diagrams from flat hierarchical data, with auto-formatting and interactive collapse/expand functionality.


Helps users understand tree structures that are diﬃcult to interpret in other formats. Also a tool to transforms CSV data with
hierarchical relationships into interactive, visual tree diagrams.

## Branches

<b>Master</b>: Features/functions that are have reached a good milestone point.

*All other branches are features/functions in development*

<b>basic-d3-tree</b>: In development.

<b>feature/csv-upload</b>: In development.

## Features
- Hover for definition
- Drag + Zoom
- Example dataset gallery
- CSV upload & processing for custom diagram generation
- Automatic formatting 
- Breadcrumb Trail
- Export PNG
- Hover eﬀects and animations

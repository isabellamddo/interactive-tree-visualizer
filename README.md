# Interactive Flowchart Visualizer
A web application that automatically generates interactive flowchart diagrams from flat hierarchical data, with auto-formatting and interactive collapse/expand functionality.

## Features
- Hover for definition
- Drag + Zoom
- Example dataset gallery
- CSV upload & processing for custom diagram generation
- Automatic formatting 
- Breadcrumb Trail
- Export SVG
- Hover eï¬€ects and animations

## How to Use
1. Open a terminal
2. Type `npm start` then press enter
3. The Interactive Tree Visualizer will open in a local host tab on your default web browser
